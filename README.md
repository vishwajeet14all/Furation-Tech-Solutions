## Tech Stack

**Client:** Node, Express, MongoD, Postman

**Server:** Node, Express


## API Reference

#### Get all items

```http
  GETALL /items
  POST /items 
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `/items` | `string` | **Required**. http://localhost:8000/items |
| `/items` | `string` | **Required**. http://localhost:8000/items |


#### Get item

```http
  GET /items/:id  
  PUT /items/:id
  DELETE /items/:id
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `:id`      | `string` | **Required**. http://localhost:8000/items/649153eb7cb10c6534c36c71 |
| `:id`      | `string` | **Required**. http://localhost:8000/items/649153eb7cb10c6534c36c71 |
| `:id`      | `string` | **Required**. http://localhost:8000/items/649153eb7cb10c6534c36c71 |
